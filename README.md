## Simple Demo Clinic API APP

Application build with:
* java 17, 
* maven, 
* hibernate,
* H2 database
* lombok, 
* mapstruct, 
* swagger, 
* angular.

Main http address:
    
Swagger:

    http://localhost:8080/swagger-ui/index.html

Angular front end with available basic CRUD operations
